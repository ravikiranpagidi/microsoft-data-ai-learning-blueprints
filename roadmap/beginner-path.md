# Beginner Path

## Goal

Build your first end-to-end Microsoft Fabric Data Engineering solution.

## Learning Steps

1. Understand what Microsoft Fabric is.
2. Create a Fabric workspace.
3. Create a Lakehouse.
4. Upload sample CSV files.
5. Run Bronze ingestion.
6. Run Silver transformation.
7. Run Gold dimensional model.
8. Query using the SQL analytics endpoint.
9. Connect Power BI.
10. Explain the architecture in your own words.

## Success Criteria

- You can explain OneLake, Lakehouse, notebooks, pipelines, Delta tables, SQL analytics endpoint, and Power BI consumption.
- You can run the notebooks in order.
- You can answer the core business questions from the sample data.
