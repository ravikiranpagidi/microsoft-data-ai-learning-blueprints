# Hands-On Practice Tasks

## Beginner Tasks

1. Create a Fabric workspace.
2. Create a Lakehouse.
3. Upload the five sample CSV files.
4. Run the Bronze ingestion notebook.
5. Query a Bronze table from the Lakehouse table view.
6. Run the Silver notebook and validate row counts.
7. Run the Gold notebook and inspect the star schema.

## Intermediate Tasks

1. Add a new transaction row and rerun the pipeline manually.
2. Add a duplicate customer row and confirm the duplicate check catches it.
3. Add a new SQL view for product category summary.
4. Add a DAX measure for rejected transaction count.
5. Create a Power BI model with all Gold relationships.

## Advanced Tasks

1. Parameterize notebooks for Dev, Test, and Prod.
2. Add incremental ingestion logic for transactions.
3. Add a pipeline activity that stops semantic model refresh when data quality fails.
4. Add row-level security by branch region.
5. Create a release note using the governance and CI/CD checklists.
