# Interview Guide

This guide helps learners prepare for Microsoft Fabric Data Engineering interviews and architecture discussions.

## Files

| File | Purpose |
| --- | --- |
| fabric-data-engineering-questions.md | Beginner and intermediate concept questions |
| scenario-based-questions.md | Real-world architecture and troubleshooting scenarios |
| architecture-discussion-points.md | Talking points for design interviews |
| hands-on-practice-tasks.md | Practical exercises to build confidence |

## How to Use

Read each question, answer it out loud, then compare your response with the strong interview-ready response. Practice explaining trade-offs, not just definitions.
