# Governance

This folder documents the governance model for the Fabric blueprint.

## Files

| File | Purpose |
| --- | --- |
| access-control-model.md | Role-based access pattern by layer |
| data-classification.md | Classification template and examples |
| pii-handling.md | PII handling, masking, and minimization guidance |
| naming-standards.md | Workspace, Lakehouse, table, pipeline, and measure naming |
| data-ownership-model.md | Owner, steward, engineer, and consumer responsibilities |
| governance-checklist.md | Production readiness checklist |

## Principle

Make the governed path the easiest path. Business users should consume certified Gold tables, SQL views, and semantic models instead of raw engineering assets.
