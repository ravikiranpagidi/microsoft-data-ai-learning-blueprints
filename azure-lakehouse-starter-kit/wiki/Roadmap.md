# Roadmap

- Sample retail dataset
- ADF ingestion templates
- Bronze, Silver, Gold Databricks notebooks
- Delta table SQL
- DQ checks
- CI validation
- Beginner-friendly docs

## Version 2: Enterprise Readiness

- Parameterized deployment examples
- Databricks Asset Bundles
- Unity Catalog grant scripts
- Incremental merge examples

## Version 3: Advanced Patterns

- Change data capture
- Streaming ingestion
- Lakehouse monitoring
- Cost observability
- ML feature tables
- Microsoft Purview integration
- Multi-domain data product examples
