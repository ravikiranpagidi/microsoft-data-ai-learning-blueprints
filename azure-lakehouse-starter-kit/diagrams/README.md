# Diagrams

This folder contains diagram source files for the Azure Lakehouse Starter Kit.

## Included Diagrams

| File | Purpose |
| --- | --- |
| `lakehouse-architecture.excalidraw` | End-to-end Azure Lakehouse architecture from source systems to Power BI and AI consumption |
| `medallion-flow.excalidraw` | Bronze, Silver, and Gold flow for Retail Customer Analytics |
| `adf-databricks-orchestration.excalidraw` | Azure Data Factory orchestration pattern for Databricks notebooks |
| `architecture.png` | Lightweight generated visual preview for GitHub and social sharing |

## Recommended Usage

- Use Mermaid diagrams in markdown for version-friendly documentation.
- Use Excalidraw files for presentation-friendly visuals.
- Keep diagram labels aligned with folder names and notebook names in this starter kit.

## Diagram Design Notes

The diagrams intentionally use Microsoft Azure service names:

- Azure Data Factory
- Azure Data Lake Storage Gen2
- Azure Databricks
- Delta Lake
- Unity Catalog
- Power BI
- Microsoft Purview

For official Microsoft icons, download the current Azure architecture icon set from Microsoft and replace the simple Excalidraw blocks with those icons in presentation copies.
