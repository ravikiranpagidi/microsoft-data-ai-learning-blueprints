# Fabric Architecture Icons

This folder contains selected SVG icons used by the architecture documentation.

The icons are sourced from the public <code>@fabric-msft/svg-icons</code> package, which Microsoft documents as the Fabric SVG icon package for platform extension development and architecture material.

## Usage Guidance

- Use the icons to illustrate Microsoft Fabric product and item concepts.
- Keep a text label close to each icon.
- Do not crop, flip, rotate, distort, recolor, or reshape the icons.
- Do not use Microsoft product icons to represent your own product or service.
- For the latest official terms, review Microsoft Learn: https://learn.microsoft.com/fabric/fundamentals/icons
