# Learning Roadmap

This roadmap guides learners from first Fabric concepts to enterprise-ready patterns.

## Paths

| Path | File | Audience |
| --- | --- | --- |
| Beginner | beginner-path.md | New Fabric learners |
| Intermediate | intermediate-path.md | Builders with basic lakehouse experience |
| Advanced | advanced-path.md | Engineers and architects preparing for enterprise use |
| 30-Day Plan | 30-day-learning-plan.md | Beginners who want a day-by-day path |
| 90-Day Growth Plan | 90-day-fabric-data-engineer-growth-plan.md | Professionals building production-ready Fabric skills |

## Recommended Use

- Use the 30-day plan when learning Fabric for the first time.
- Use the 90-day plan when preparing for professional Fabric data engineering work.
- Use the beginner/intermediate/advanced paths as skill-area summaries.
