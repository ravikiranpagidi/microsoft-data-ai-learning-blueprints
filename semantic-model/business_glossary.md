# Business Glossary

## Entities

| Term | Definition |
| --- | --- |
| Customer | A person or business entity with a banking relationship. |
| Account | A financial account owned by a customer and linked to a banking product. |
| Product | A banking product such as checking, savings, credit card, or investment account. |
| Transaction | A monetary event posted, rejected, or pending against an account. |
| Branch | A physical or operational banking location. |
| Customer Segment | Business grouping used to analyze customers by value, behavior, or relationship type. |

## Metrics

| Metric | Definition |
| --- | --- |
| Active Customer Count | Distinct customers with active status. |
| Transaction Count | Distinct transaction IDs in the selected filter context. |
| Posted Transaction Count | Distinct posted transactions only. |
| Total Transaction Amount | Sum of absolute transaction amounts. |
| Average Transaction Amount | Total transaction amount divided by transaction count. |
| Account Count | Distinct accounts. |
| Open Account Count | Distinct accounts with open status. |
| Total Current Balance | Sum of account current balances. |
| Customer Segment Count | Distinct customer segments. |
