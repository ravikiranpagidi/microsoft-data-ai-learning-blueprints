# CI/CD

This folder explains how to manage Microsoft Fabric changes across Dev, Test, and Prod.

## Files

| File | Purpose |
| --- | --- |
| git-integration-guide.md | What Git integration is and how to use it safely |
| deployment-pipeline-guide.md | Promotion flow and gates |
| dev-test-prod-strategy.md | Environment strategy |

## Principle

Version what should be reviewed. Configure what should vary by environment. Promote only what has passed validation.
