# Release Checklist

## Code and Configuration

- [ ] Pull request reviewed.
- [ ] Notebooks committed.
- [ ] SQL scripts committed.
- [ ] Pipeline templates updated.
- [ ] Data quality rules updated.
- [ ] Documentation updated.
- [ ] No secrets committed.

## Validation

- [ ] Bronze ingestion completed.
- [ ] Silver transformation completed.
- [ ] Gold dimensional model completed.
- [ ] Data quality passed.
- [ ] SQL views executed successfully.
- [ ] Power BI semantic model refreshed.
- [ ] Key business metrics reconciled.

## Governance

- [ ] PII review completed.
- [ ] Access groups reviewed.
- [ ] Data owner approval captured.
- [ ] Release notes prepared.

## Deployment

- [ ] Deployed to Test.
- [ ] UAT completed.
- [ ] Deployed to Prod.
- [ ] Production refresh monitored.
- [ ] Stakeholders notified.
