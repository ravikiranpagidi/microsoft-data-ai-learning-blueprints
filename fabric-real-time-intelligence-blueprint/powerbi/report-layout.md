# Power BI Report Layout

## Executive Operations Overview

Visuals:

- KPI cards for Total Shipments, Delay Rate, SLA Breach Count, Active Vehicles, Temperature Breach Count.
- Line chart for On-Time Delivery Percentage by date.
- Bar chart for Average Delay Minutes by region.
- Table for top delayed routes.

## Real-Time Shipment Monitoring

Visuals:

- Shipment status distribution.
- Delayed shipment detail table.
- Region slicer.
- Route slicer.
- Map visual for latest vehicle location if available.

## Delay And SLA Analysis

Visuals:

- Delay Rate by route.
- SLA Breach Count by customer region.
- Average Delay Minutes trend.
- Route delay leaderboard.

## Vehicle Health

Visuals:

- Vehicle Alert Count by vehicle.
- Max Engine Temperature by vehicle.
- Harsh Braking Count trend.
- Low fuel vehicle table.

## Warehouse Conditions

Visuals:

- Temperature Breach Count by warehouse.
- Average temperature by zone.
- Door Open Count by warehouse.
- Condition breach detail table.

## Regional Performance

Visuals:

- Total shipments by region.
- On-Time Delivery Percentage by region.
- Average Delay Minutes by region.
- SLA breach distribution.
