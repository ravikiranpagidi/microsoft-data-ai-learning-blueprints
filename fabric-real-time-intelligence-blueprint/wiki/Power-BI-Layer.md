# Power BI Layer

## Purpose

Explain how Power BI consumes curated logistics data.

## Report Pages

- Executive Operations Overview.
- Real-Time Shipment Monitoring.
- Delay and SLA Analysis.
- Vehicle Health.
- Warehouse Conditions.
- Regional Performance.

## Measures

- Total Shipments.
- Delayed Shipments.
- Delay Rate.
- Average Delay Minutes.
- SLA Breach Count.
- Active Vehicles.
- Temperature Breach Count.
- On-Time Delivery Percentage.

## Best Practices

- Use curated Gold tables.
- Define metric ownership.
- Hide technical fields.
- Keep dashboard and Power BI roles clear.
