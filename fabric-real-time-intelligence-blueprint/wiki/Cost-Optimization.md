# Cost Optimization

## Purpose

Explain cost and capacity considerations for the blueprint.

## Cost Drivers

- Event volume.
- Event size.
- Eventhouse retention.
- Query frequency.
- Dashboard refresh patterns.
- Number of Activator rules.
- Lakehouse storage.
- Power BI model design.

## Practical Controls

- Start demos with low event volume.
- Use time filters in KQL.
- Use summary tables for Power BI.
- Define retention policies.
- Monitor capacity during load tests.
- Avoid broad exploratory queries during live demos.
