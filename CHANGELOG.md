# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-20

### Added

- Initial Microsoft Fabric Data Engineering blueprint.
- Retail Banking Customer Analytics sample domain.
- End-to-end documentation for Fabric concepts, architecture, governance, CI/CD, and best practices.
- Sample CSV datasets for customers, accounts, products, transactions, and branches.
- Fabric-compatible PySpark notebooks for setup, Bronze ingestion, Silver transformation, Gold modeling, data quality, Delta optimization, and Power BI-ready views.
- SQL scripts for Gold views, business metrics, validation, and Power BI consumption.
- YAML-driven data quality framework.
- Semantic model guidance, DAX measures, and business glossary.
- Architecture Decision Records.
- Interview guide and learner roadmap.
- Community contribution planning assets.
