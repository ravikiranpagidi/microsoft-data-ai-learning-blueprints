# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-05-20

### Added

- Fabric Decision Guide covering Lakehouse vs Warehouse, Notebook vs Pipeline, Dataflow Gen2 vs Notebook, Shortcut vs Copy, Gold table vs SQL view, Star schema vs Wide table, and Direct Lake vs Import mode.
- Real-world architecture patterns for small teams, enterprise teams, data products, self-service BI, and AI-ready lakehouses.
- Common Enterprise Mistakes guide with corrections.
- Fabric Project Checklist for workspace setup through production readiness.
- 30-Day Learning Plan and 90-Day Fabric Data Engineer Growth Plan.
- Incremental load and operational monitoring notebooks.
- Advanced analytics and operational monitoring SQL scripts.
- GitHub issue templates and pull request template.
- README badges and stronger beginner explanations.
- Direct Lake semantic model ADR.

## [0.1.0] - 2026-05-20

### Added

- Initial Microsoft Fabric Data Engineering blueprint.
- Retail Banking Customer Analytics sample domain.
- End-to-end documentation for Fabric concepts, architecture, governance, CI/CD, and best practices.
- Sample CSV datasets for customers, accounts, products, transactions, and branches.
- Fabric-compatible PySpark notebooks for setup, Bronze ingestion, Silver transformation, Gold modeling, data quality, Delta optimization, and Power BI-ready views.
- SQL scripts for Gold views, business metrics, validation, and Power BI consumption.
- YAML-driven data quality framework.
- Semantic model guidance, DAX measures, and business glossary.
- Architecture Decision Records.
- Interview guide and learner roadmap.
- Community contribution planning assets.
